Рекомендации по составлению промптов

Лучшие практики для создания эффективных промптов архитектурной визуализации

Будьте конкретны

Используйте точные описания вместо общих фраз. Вместо "красивое здание" лучше указать "современное здание из стекла и бетона с минималистичным фасадом". 

"современное здание из стекла и бетона с минималистичным фасадом"

Детализируйте освещение

Указывайте время суток, направление света и характер теней. Например: "закатное освещение, солнце слева, длинные драматические тени". 

"закатное освещение, солнце слева, длинные драматические тени"

Описывайте материалы

Детализируйте текстуры и материалы поверхностей. Укажите тип покрытия земли, отделку фасада, материалы мебели и ландшафта.

Учитывайте сезон и погоду

Сезон и погода влияют на атмосферу сцены. Зимний снег, осенние листья, летняя зелень — все это создает нужное настроение.

Детализируйте людей

Если добавляете людей, опишите их действия и одежду. Например: "взрослые идут с детьми, дети играют, все одеты по погоде". 

"взрослые идут с детьми, дети играют, все одеты по погоде"

Указывайте, чего избегать

Используйте поле "avoid" для исключения нежелательных элементов: "избегай сдвига камеры, изменения угла, артефактов 3D-рендера". 

"избегай сдвига камеры, изменения угла, артефактов 3D-рендера"

Технические параметры

Указывайте параметры камеры и качество: "f/8, ISO 100, архитектурная фотография, 8K, гиперреалистично". 

"f/8, ISO 100, архитектурная фотография, 8K, гиперреалистично"

Для Reconstruct: сохраняйте геометрию

Всегда подчеркивайте сохранение исходной геометрии скетча. Референс используется только для материалов и стиля, не для изменения структуры.

Дополнительные советы

  • Начинайте с простых сценариев и постепенно усложняйте
  • Используйте готовые сценарии как основу и адаптируйте под свои задачи
  • Экспериментируйте с комбинациями параметров для уникальных результатов
  • Сохраняйте успешные конфигурации для повторного использования
